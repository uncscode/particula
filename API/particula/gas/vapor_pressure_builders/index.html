    particula.gas.vapor_pressure_builders¶

    vapor_pressure_builders ¶

    Builders to create vapor pressure models for gas species.

    This module provides builder classes for Antoine, Clausius-Clapeyron, constant, and WaterBuck vapor pressure strategies. Each builder follows the same workflow:

    1. Configure coefficients or parameters using dedicated methods.
    2. Validate required parameters.
    3. Return the corresponding vapor pressure strategy object.
    References
    • "Antoine Equation," Wikipedia
    • "ClausiusClapeyron Relation," Wikipedia
    • "Vapor Pressure," Wikipedia

    AntoineVaporPressureBuilder ¶

    AntoineVaporPressureBuilder()

    Bases: BuilderABC

    Builder class for AntoineVaporPressureStrategy.

    It allows setting the coefficients 'a', 'b', and 'c' separately and then building the strategy object. Follows the general form of the Antoine equation in

    log(P) = a  b / (T  c)

    Attributes:

    • - (a)

      Coefficient "a" of the Antoine equation (dimensionless).

    • - (b)

      Coefficient "b" (in Kelvin).

    • - (c)

      Coefficient "c" (in Kelvin).

    Methods: - set_a : Set the coefficient "a" of the Antoine equation. - set_b : Set the coefficient "b". - set_c : Set the coefficient "c". - build : Validate parameters and return an AntoineVaporPressureStrategy.

    Example
    AntoineVaporPressureBuilder
    strategy = (
    AntoineVaporPressureBuilder()
    .set_a(8.07131)
    .set_b(1730.63)
    .set_c(233.426)
    .build()
)
    AntoineVaporPressureBuilder with units
    strategy = (
    AntoineVaporPressureBuilder()
    .set_a(8.07131)
    .set_b(1730.63, "K")
    .set_c(233.426, "K")
    .build()
)
    References
    • Equation: log10(P_mmHG) = a - b / (Temperature_K - c) (Reference: https://en.wikipedia.org/wiki/Antoine_equation)
    • "Vapor Pressure," Wikipedia
    • "Atmospheric Pressure Unit Conversions," Wikipedia

    ClausiusClapeyronVaporPressureBuilder ¶

    ClausiusClapeyronVaporPressureBuilder()

    Bases: BuilderABC

    Builder class for ClausiusClapeyronStrategy. This class facilitates setting the latent heat of vaporization, initial temperature, and initial pressure with unit handling and then builds the strategy object.

    The ClausiusClapeyron relation can be approximated as:

    • dP / dT = (L / (R T))

    Methods: - set_latent_heat : Set latent heat in J/mol (or convertible units). - set_temperature_initial : Set initial temperature in K (or convertible units). - set_pressure_initial : Set initial pressure in Pa (or convertible units). - build : Validate parameters and return a ClausiusClapeyronStrategy.

    Example
    ClausiusClapeyronVaporPressureBuilder
    strategy = (
    ClausiusClapeyronVaporPressureBuilder()
    .set_latent_heat(2260)
    .set_temperature_initial(373.15)
    .set_pressure_initial(101325)
    .build()
)
    ClausiusClapeyronVaporPressureBuilder with units
    strategy = (
    ClausiusClapeyronVaporPressureBuilder()
    .set_latent_heat(2260, "J/mol")
    .set_temperature_initial(373.15, "K")
    .set_pressure_initial(101325, "Pa")
    .build()
)
    References
    • Equation: dP/dT = L / (R * T^2) https://en.wikipedia.org/wiki/Clausius%E2%80%93Clapeyron_relation
